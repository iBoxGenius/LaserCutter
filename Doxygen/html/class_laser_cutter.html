Detailed Description

Main Class for the whole application.

Handles most of the funcionalities of from the user's side. Implements the OpenCV methods and controls the workflow of the application.

◆ CameraMarkerDetection()

bool LaserCutter::CameraMarkerDetection ( )
private

Detection of ArucoMarkers from camera input.

Detects the four markers from the selected camera. If the markers are not detected after five seconds, returns false. Also stores the positions of each of the [0] corneres of the Arucos to m_corners variable

Returns
Returns true if the all four markers are found, otherwise false

◆ CheckSvgsPositions()

bool LaserCutter::CheckSvgsPositions ( )
private

Checks the positions of the SvgLabel instance whether they are within the bounds of the material.

Takes an image from the camera input and applies binary thresholding (inverse, non-inverse). Grabs the image from the image_destination DragWidget instance with black and white background. Applies the same thresholding. The white_label is then negated. ANDs the corresponding images and compares them with original(of the camera frame thresholded images). If the Images are equal, objects are within the bounds of the material

Returns
Returns true if images are within the bounds of the material, otherwise false.

◆ Initialize

void LaserCutter::Initialize ( )
privateslot

Initialization of the app.

Calls the InitCutterInfo() method to obtain the neccessary paramters. Hands over the camera transmission to the CameraMarkerDetection() method, and later to the PerspectiveTransform() method. Restarts the timers in place. and initializes the spinBox values. If marker detection succeeds, calls SetupUiFrameSize(), GetTransferRatio() methods, creates the Temp folders, calls CopySvgsFromDir(). Connects the neccessary signals.

◆ RotateLabel()

void LaserCutter::RotateLabel ( )
private

RotateLabel Rotates an SvgLabel.

Rotates the currently selected SvgLabels. Uses the GetMinimumBoundingRect method to calculate the dimensions of the bounding box. Moves the SvgLabel by the offset regarding the rotation. Utilizes QTransform (affine transformation) to rotate the SvgLabel.
