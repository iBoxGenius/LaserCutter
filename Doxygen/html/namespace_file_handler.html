Creates the LaserCutter surface SVG.

Creates the LaserCutter surface SVG with all the images placed correctly placed. Function creates a copy of the reference surface file and copies all the vector properties (rect, circle etc.) of the placed images to the final SVG file, which is later used as a source for G-Code generation.

Parameters

Returns
Returns false if the process failed, otherwise true
